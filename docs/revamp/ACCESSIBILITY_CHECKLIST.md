# WCAG 2.1 AA Accessibility Checklist

## Completed
- [x] Skip-to-content link (layout)
- [x] Visible focus states (`:focus-visible`)
- [x] `prefers-reduced-motion` respected (animations)
- [x] Form labels associated with inputs
- [x] ARIA labels on icon buttons (menu, close)
- [x] Semantic HTML (main, nav, section, header, footer)

## To Verify
- [ ] Color contrast 4.5:1 (normal text), 3:1 (large text); 3:1 for UI components and graphical objects (non-text contrast)
- [ ] All interactive elements keyboard navigable
- [ ] No keyboard traps
- [ ] Alt text on all images
- [ ] Error messages announced to screen readers
- [ ] Video captions (if any)
